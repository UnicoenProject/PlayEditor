Jasminを使ってSwiftを学ぼう

この度は、私たちの科学実験にご協力いただきまして、誠にありがとうございます。

私たちは、言語変換を活用したプログラミング言語の学習方法について研究しています。
普段お使いのプログラミング言語と学習中の言語を比較することで、言語学習を支援しようとしています。

今回は、言語変換機能を取り入れた学習環境をWebアプリとして提供いたします。
本実験では、本学習環境を使って、Swiftの学習に取り組んでいただきます。

以下で、本学習環境の使い方について説明いたします。
なお、本学習環境はFirefoxとGoogle Chromeで動作確認をしており、どちらかの環境でのご利用をおすすめします。

本ページの最下部にある「START」ボタンを押すと、以下の図のようなSwiftの練習問題が表示されます。
練習問題は前半と後半に分かれており、前半はリーディング問題、後半はライティング問題になります。

最下部にはJavaプログラムをSwiftプログラムに変換できる変換器があります。
変換器には1つのJavaクラスしか入力することができませんので、ご注意ください。
つまり、クラスの定義から始まって、フィールドとメソッドの定義がその中にあり、さらにメソッドの中にステートメントがある形を想定しています。

例えば、

class Klass {
    for (int i = 0; i < 5; i++) {
    }
}
上記のコードはエラーとなり、正確にSwiftプログラムに変換できません。

一方、 

class Klass {
    void method() {
        for (int i = 0; i < 5; i++) {
        }
    }
}
上記コードは正しく変換できるJavaプログラムです。

ライティング問題では、コーディング環境を用意して取り組んで頂くことを想定しています。
MacユーザーはXCodeを開き、メニューのFile -> New -> Playgroundで.playgroundファイルを作成して、Swiftコードを書いて動かすことができます。
それ以外のユーザはオンラインコンパイラである http://www.runswiftlang.com または http://swiftstub.com/ などをご利用いただけます。
